All comments below are about very minor potential issues that you may choose to
address in some way - or ignore - as you see fit. Some were flagged by

### Grammar/style

#### Section 3.1, paragraph 1
```

                                ^^^^^^^^^
```
Consider simply using "of" instead.

#### Section 5, paragraph 1
```
ation This document is the result of an comprehensive inventory conducted by
                                    ^^
```
Use "a" instead of "an" if the following word doesn't start with a vowel sound,
e.g. "a sentence", "a university".

#### Section 6, paragraph 1
```
ins are highly likely to be obsolete and this document is intended to formal
                                    ^^^^
```
Use a comma before "and" if it connects two independent clauses (unless they
are closely connected and short).

#### Section 6, paragraph 1

Specify a number, remove phrase, use "a few", or use "some".
