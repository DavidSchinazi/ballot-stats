draft-rprice-ups-management-protocol has been presented to the ISE for
publication as an Informational RFC on the Independent Stream.

==Purpose==

This protocol largely documents an existing protocol known as NUT, a
management protocol for Uninterrupted Power Supply (UPS) systems.
This document clarifies security considerations, changes some
terminology, and provides for proper versioning.

== History==

NUT has existed for several decades.  The author approached the ISE
to publish this document on the 6th of March, 2021, and has undergone
substantial clarification since.

==Non-IETF Work==

This work supports the NUT open source project.

==Security Considerations==

A new security considerations section covers the protocol.  The
protocl permits as a policy choice clear text communications, and
specifies several methods to encrypt communication.  The clear text
communication is based on existing use.

==IANA==

This specification requests an update of contact information for the
NUT protocol.

==Reviews==

Apart from ISE reviews, members of the NUT community have reviewed the
work.
