Subject: Last Call:  (GeoJSON Text Sequences) to Proposed Standard


The IESG has received a request from the Geographic JSON WG (geojson) to
consider the following document:
- 'GeoJSON Text Sequences'
  as Proposed Standard

The IESG plans to make a decision in the next few weeks, and solicits
final comments on this action. Please send substantive comments to the
ietf@ietf.org mailing lists by 2017-01-24. Exceptionally, comments may be
sent to iesg@ietf.org instead. In either case, please retain the
beginning of the Subject line to allow automated sorting.

Abstract


  A proposed standard for geographic data that can be parsed and
  produced incrementally.




The file can be obtained via
https://datatracker.ietf.org/doc/draft-ietf-geojson-text-sequence/

IESG discussion can be tracked via
https://datatracker.ietf.org/doc/draft-ietf-geojson-text-sequence/ballot/


No IPR declarations have been submitted directly on this I-D.
