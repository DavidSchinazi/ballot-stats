This document updates _CCNx messages in TLV Format_ [RFC8609] to specify a logarithmic encoding for selected time values.

This document makes no semantic changes to [RFC8569], nor to any of the security properties of the message encodings of [RFC8609], and hence has the same security considerations as those two existing documents.

With respect to IANA considerations, based on the approach of integration, certain TLV registries from [RFC8609] need to be updated.

This draft has first been submitted as an individual contribution in 2019-11 and has been adopted as a RG item in 2022-04. It is an ICNRG product and has been reviewed by the group. The RG last call has not led to any additional change requests.
