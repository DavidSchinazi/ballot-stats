The IANA Functions Operator understands that upon approval of this document, we'll need to complete two actions.

First, in the SMI Security for S/MIME Module Identifier (1.2.840.113549.1.9.16.0) registry on the Structure of Management Information (SMI) Numbers (MIB Module Registrations) registry page located at:

https://www.iana.org/assignments/smi-numbers/

a single new registration will be made:

Decimal: [ TBD-at-Registration ]
Description: id-mod-hkdf-oid-2019
Reference: [ RFC-to-be ]

Second, in the SMI Security for S/MIME Algorithms (1.2.840.113549.1.9.16.3) registry also on the Structure of Management Information (SMI) Numbers (MIB Module Registrations) registry page located at

https://www.iana.org/assignments/smi-numbers/

three new registrations are to be made:

Decimal: [ TBD-at-Registration ]
Description: id-alg-hkdf-with-sha256
Reference: [ RFC-to-be ]

Decimal: [ TBD-at-Registration ]
Description: id-alg-hkdf-with-sha384
Reference: [ RFC-to-be ]

Decimal: [ TBD-at-Registration ]
Description: id-alg-hkdf-with-sha512
Reference: [ RFC-to-be ]

Because all four registrations are to be made in Specification Required registries, we're asking the secondary designated expert to review and approve. (The primary expert is the author of this document.)

The IANA Functions Operator understands that these are the only actions required by this document. They will not be completed until the document is approved for publication.

Thank you,

Abstract


  RFC 5869 specifies the HMAC-based Extract-and-Expand Key Derivation
  Function (HKDF) algorithm.  This document assigns algorithm
  identifiers to the HKDF algorithm when used with three common one-way
  hash functions.




The file can be obtained via
https://datatracker.ietf.org/doc/draft-housley-hkdf-oids/

IESG discussion can be tracked via
https://datatracker.ietf.org/doc/draft-housley-hkdf-oids/ballot/


No IPR declarations have been submitted directly on this I-D.
